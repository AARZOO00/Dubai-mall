# The Dubai Mall — Interactive Sales Deck

A browser-based high-end sales experience built as a pitch deck for The Dubai Mall.

## What’s Included

- `index.html` — Core interactive deck with cinematic video hero, non-linear navigation, immersive section storytelling, and conversion-focused CTAs.
- `events.html` — Detailed event hosting module describing event capabilities, venue services, and booking flow.
- `sponsorship.html` — Sponsorship and partnership module with tiered packages and audience data.
- `clientele.html`, `maisons.html`, `vip.html` — Supporting luxury and guest experience pages.

## Why This Matches the Brief

- **Video-first hero**: Cinematic autoplay/background video with modal video play.
- **Interactive navigation**: Top nav links allow non-linear exploration across overview sections.
- **Story beats covered**: Why this property, retail, luxury, dining, attractions, events, sponsorship, and contact.
- **Expandable architecture**: Dedicated sub-pages and interactive leasing/sponsorship modules demonstrate growth paths.
- **Luxury UI**: Minimal chrome, premium typography, polished hover interactions, and rich cinematic styling.
- **Responsive**: Desktop and tablet-friendly, with adaptions for smaller screens.

## How to Run

1. Open `index.html` in a browser.
2. For deployment, publish the repository to GitHub and enable GitHub Pages on the repository root.
3. Alternatively, deploy with Netlify or Vercel by pointing at the repository root.

## Notes

- The deck is fully self-contained and built with plain HTML, CSS, and JavaScript.
- Visual assets are sourced from publicly available imagery and video references.
- The structure is designed for an immersive sales presentation and standalone exploration.
